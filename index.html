Porfolio project description, CMP342 Instr Diop due date 12/09/2011


As a job seeker, it is necessary to express your abilities to impress prospective employers.
A portfolio demonstrates your creativity and may be used as a plataform for self-expression, giving
employers an insight of who you are beyond a few minutes of an interview. Electronic-portfolio
can be maintaned dynamically over time, permits varying degrees of audience access, so the
same portfolio might be used for multiple purposes. Use this opportunity to impress your future
with your creativity.

Required

  1. Project site should have a minimum of 6 pages with HTML5.
  2. index.html should include the following:
    • A header.inc or header.php (use for all pages)
    • Footer.inc/php (use for all pages)
    • Sidebar
    • Two(2) articles
    *Header.inc should include a changeable navigation bar and a full size image
  3. Create an interactive page:
    • Using either a frame or a Canvas or Scalable Vector Graphics(SVG)
    • Embed Client-side scripts withing an HTML5 documents.
    • Server-side scripting a user's request is verified by runing a script directly on the web server
  4. All html should be linked to a *.CSS for styling.
  5. Create a link to a social media
  6. Email login in.
